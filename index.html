EventSurf - Android Application

This is an Android project I created with a friend in our fourth semester at Fontys. The app combines the functionality of CouchSurfing, BlaBlaCar and a friend finder app revolving around music festivals.
The process consisted of concepting, creating wireframes, mockups, an interactive prototype and in the end, we developed the app's some key functionality and the layout in native Android. We store data in Firebase and use SongKick's API to retreive event info.
